## Getting started

`$ yarn add @busfor/react-native-touchable`

## Usage

```javascript
import { Touchable } from '@busfor/react-native-touchable'
```

```jsx
<Touchable
  onPress={() => console.log('It works!')}
  // ...other TouchableOpacity props
>
  ...
</Touchable>
```
